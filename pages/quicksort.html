Visualising the Quick Sort algorithm

What is Quick Sort?

Quick Sort is a highly efficient sorting algorithm, developed by British computer scientist Tony Hoare in 1959. It works by selecting a 'pivot' element and creating two sub-arrays with all the elements bigger than the pivot on one, and smaller on the other. The sub-arrays are then sorted recursively.

This algorithm is quite efficient for large data sets as its average and worst-case complexities are O(nLogn) and O(n^2) respectively, although the worst case scenarios are rare.

How it works:

  1. Pick an element from the array. This will be the pivot. There are many ways to choose the pivot: selecting the first element, the last, randomly, median... In this example I will choose the first element.
  2. Partition: reorder the array so that all the elements with values less than the pivot come before the pivot while all the elements with values greater than the pivot come after it. After the partitioning, the pivot will be in its final position.
  3. Recursively apply the above steps to each sub-array of elements.
