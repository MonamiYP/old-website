A small intorduction to machine learning

Machine learning is a branch of artificial intelligence which focuses on the use of data and algorithms to imitate the way humans learn, graually improving its accuracy. It builds a model based on sample data, known as trining data, in order to make predictions or decisions without being explicitly programmed to do so.


In this article I will explore a few of the most common and simple algorithms used to predict and model data. Before that, however, we have to know a bit abou the three main types of algorithms: supervised, unsupervised and reinforcement learning.

Types of Machine Learning Algorithms

  1. Supervised learning - creates a model to predict an outcome based on labeled data. It is told the correct output and compares its own output which informs the subsequent steps, adjusting itself along the way. For example, the machine gets an image of a dog and it has to predict wether it's a dog or a cat.

  2. Unsupervised learning - determines patterns in unlabeled data, often used to create groups and clusters. Since the data is unlabeled, the machine must discover unknown patterns and structures. For example, getting images of animals and classifying them into clusters learned automatically based on the data.

  3. Reinforcement learning - provides training feedback using a reward an punishment mechanism; the Agent can interact with the environment and can try a variety of methods to reach an outcome. It is then rewarded or punished when it reaches a desirable or undesirable state.

Today I will explore a few supervised machine learning algorithms: linear regression, logistic regression, decision tress and K-nearest neighbours. This is just an overview of how they work; I will not include any code here. More in depth analysis of these algorithms are coming soon. Let's get started...

Linear Regression

This is the most basic type of regression, where a set of input variables (x) are used to determine an output (y). The relatonship between x and y are expressed as an equation in the form y=mx+c, thus the goal is to find the values of m and c which best quantifies this relationship.

Logistic Regression

Logistic regression predicts discrete values, best suited for binary classification: data sets where the output is either 0 or 1. For example, whether a student has passed or failed. The logistic function, or sigmoid function is y(x) = 1 / (1 + e^(-x)), which output the probability of y. The output y is generated using this function and a threshold is then applied to force this probability into a binary classification.

K-nearest Neighbours

Unlike the other algorithms which split the data into a training set and a test set, KNN uses the entire data set as a training set. As the name suggests, it considers K nearest data points to predict the class or continuous value for a new datapoint.
